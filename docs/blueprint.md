# **App Name**: BZION Hub Digital

## Core Features:

- Product Catalog: Display a comprehensive catalog of FMCG products with filtering and sorting options based on categories, brands, and other relevant attributes.
- Request for Quote (RFQ): Enable users (retailers, wholesalers, institutions) to request quotes for specific products and quantities, streamlining the procurement process.
- Supplier Directory: AI tool which can classify suppliers and assist users with their decision process.
- Account Management: Provide user accounts with role-based access control for retailers, wholesalers, and suppliers to manage profiles, orders, and quotes.
- Content Creation: Create all content based on the BZION Hub Website Content document, including text for headers, footers, navigation, home page sections, about page, product pages and contact information. This includes global elements and dynamic content population.

## Style Guidelines:

- Primary color: BZION Blue (#00205B) to convey trust, stability, and corporate identity.
- Accent color: BZION Gold (#FFD600) for energy, value, and call-to-action elements.
- Background color: Light gray (#F1F5F9) to provide a clean and modern look.
- Body and headline font: 'Inter', a sans-serif for modern legibility. Note: currently only Google Fonts are supported.
- Implement a responsive grid system using Tailwind CSS for optimal viewing on different devices.
- Use lightweight icons from Lucide React for consistent and clear visual cues.
- Apply subtle animations using Framer Motion for enhanced user experience and interactivity.
